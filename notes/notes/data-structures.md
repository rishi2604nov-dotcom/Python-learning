# Python Data Structures Notes

## 1. Lists
```python
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits[0])   # apple
```

---

## 2. Tuples (Immutable)
```python
point = (10, 20)
print(point[1])   # 20
```

---

## 3. Dictionaries
```python
person = {"name": "Rishikesh", "age": 28}
print(person["name"])   # Rishikesh
```

---

## 4. Sets
```python
nums = {1, 2, 3, 3}
print(nums)   # {1, 2, 3}
```
